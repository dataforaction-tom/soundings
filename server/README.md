# soundings server

The FastAPI + MCP server for Soundings. Wraps UK open-data sources behind a small set of question-shaped tools, anchored to a single canonical geography spine, with every consented call written to a published questions corpus.

Managed with `uv`. See top-level [`README`](../README.md) for context and [`Makefile`](../Makefile) for the operator-facing commands.
